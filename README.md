## 🚀 Getting Started

### 📦 Dependency

- node v18+
- npm v9.5+
- gulp v4+

### 👉 Package Installation

```
npm install
```

### 👉 Development Command

```
npm run dev
```

### 👉 Build Command

```
npm run build
```
